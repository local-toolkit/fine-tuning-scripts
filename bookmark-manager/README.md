# MarkNest · Chrome 书签管理器

一个完全本地、基于 Chrome 原生书签的全屏管理页面。点击扩展图标即可打开。

## 功能

- 文件夹树与子文件夹聚合浏览
- 标题、网址、文件夹即时搜索（`⌘/Ctrl + K`）
- 卡片 / 列表双视图与多种排序
- 大书签库渐进式渲染：首屏只创建首批卡片，搜索仍覆盖全部数据
- 新建、内联编辑、删除和批量移动
- 最近 30 天、重复网址、失效链接智能视图
- 深色模式；布局和扫描结果本地保存
- 不需要账号，不上传书签，不使用分析服务
- 重复点击图标会复用已有管理页，不会创建多个页面实例

## 安装

1. 打开 `chrome://extensions/`。
2. 开启右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录 `bookmark-manager`。
5. 点击工具栏中的 MarkNest 图标。

修改代码后，在扩展管理页点击此扩展的刷新按钮即可看到最新版。

## 权限

- `bookmarks`：读取、新建、编辑、移动和删除 Chrome 书签。
- `storage`：保存主题、布局和最近一次失效链接扫描结果。
- `favicon`：在本地显示网站图标。
- `tabs`：复用已经打开的 MarkNest 管理页。
- `<all_urls>`：仅在用户进入“失效链接”视图时检查书签地址。

## 文件

- `manager.html / manager.css / manager.js`：完整管理页面
- `background.js`：打开页面与失效链接扫描
- `manifest.json`：Manifest V3 配置
