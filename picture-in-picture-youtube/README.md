# Picture-in-Picture Local - YouTube Button

This is a local version of Google's Picture-in-Picture Extension with an extra YouTube page button.

## Load in Chrome

1. Open `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this folder: `picture-in-picture-youtube`.

The extension still supports the toolbar action and `Alt+P`. On YouTube, it also adds a **PiP** button into the player controls.
